# Basic Calculator
This program asks the user for two numbers and performs three basic math operations: addition, subtraction, and multiplication.

## Features
- Asks the user for two numbers  
- Calculates their sum  
- Calculates their difference  
- Calculates their product  
- Prints all results to the screen  



## Learning Purpose
A simple exercise to practice:
- user input  
- converting input to integers  
- basic arithmetic  
- printing results
